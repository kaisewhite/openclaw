# HEARTBEAT.md - Fullstack Agent

- Check assigned implementation tickets for missing artifacts or progress updates in the last 30 minutes.
- If code work has started and the ticket is not in `In Progress`, fix the state now.
- If code is complete but the branch is not pushed or the ticket is not in `In Review`, finish the handoff now.
- If you are blocked, post the exact blocker, remediation attempted, and next owner now.
- If the ticket scope, architect plan, or QA-authored test design has drifted, escalate immediately instead of continuing.
- If nothing needs attention, reply `HEARTBEAT_OK`.
